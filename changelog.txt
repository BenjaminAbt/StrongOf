StrongOf - Changelog
=====================

Unreleased
----------

Added
- Source generator (`StrongOf.SourceGenerators`): generate AOT-friendly strong types from a single line.
  Marker attributes in the `StrongOf.SourceGeneration` namespace turn a `partial` class into a fully
  implemented strong type without any Expression-based factory:

      [StrongGuid]   public partial class UserId;
      [StrongString] public partial class Email;
      [StrongInt32]  public partial class Quantity;

  The generator emits the primary constructor, the base type (e.g. `StrongGuid<UserId>`) and a
  static `Create` member that satisfies the `IStrongOf<TTarget, TSelf>.Create` contract directly
  with `new TStrong(value)` - no reflection, no Expression.Compile, fully Native AOT and trim safe.
- The generator ships inside the `StrongOf` NuGet package (`analyzers/dotnet/cs/`); no extra
  package reference is required.
- New marker attributes: `StrongBoolean`, `StrongChar`, `StrongDateTime`, `StrongDateTimeOffset`,
  `StrongDecimal`, `StrongDouble`, `StrongGuid`, `StrongInt32`, `StrongInt64`, `StrongString`,
  `StrongTimeSpan`.

Diagnostics
- `STRONG001` (error): the marker attribute requires a `partial` class.
- `STRONG002` (error): nested types are not supported.
